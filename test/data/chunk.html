Getting Started

To start using Android KTX in your Android Kotlin projects, add the following to your app module's build.gradle file: 

repositories {
    google()
}

dependencies {
    // Android KTX for framework API
    implementation 'androidx.core:core-ktx:0.1'
    ...
}
